# studyBotTelegram
Бот телеграм для учебы в вузе 
Бот умеет добавлять/удалять пары, показывать расписание на неделю.

Бот будет уметь: добавлять/удалять файлы или иные материалы по дисциплинам, соответственно показывать их по запросу.
Бот будет уметь: отправлять уведомления преподавателям (в телеграмме) с некоторыми напоминаниями.
